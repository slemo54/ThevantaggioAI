# IlVantaggioAI.it — Autoblogger AI (Project Brain)

Questo repository contiene il **“cervello” del progetto**: documentazione canonica, roadmap, prompt library e regole operative per costruire e mantenere l’autoblogger AI di **IlVantaggioAI.it**.

## Start qui

- Leggi `docs/brain/INDEX.md` (mappa completa + cosa è “source of truth”)
- Incolla `docs/brain/CONTEXT_PACK.md` come contesto base per qualunque agente/LLM
- Lavora seguendo `docs/brain/AGENTS.md` (ruoli, handoff, Definition of Done)

## Obiettivo del sistema (in 1 riga)

Pipeline end‑to‑end: **News → Draft → SEO+Editing → Immagini → Publish (Next.js) → Distribuzione** con qualità editoriale e SEO avanzata.

## Convenzioni

- **Lingua**: Italiano di default (a meno di esplicita richiesta diversa).
- **Commit**: Conventional commits (`feat:`, `fix:`, `docs:`, `chore:`).
- **Decisioni**: ogni scelta “architetturale” va in `docs/brain/ADR/` (ADR).
