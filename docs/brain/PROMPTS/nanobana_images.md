## Input

- `[TOPIC]`
- `[CATEGORY]`
- `[KEY_CONCEPT]` (per diagramma mid)
- `[KEYWORD_PRIMARY]`
- `[SPECIFIC_VISUAL_FROM_ARTICLE]`

## Brand style

- Palette: `#1a7f8f` (teal), `#0d4f5f` (dark teal), `#00d4ff` (cyan), `#ffffff`, `#f0f0f0`
- Stile: moderno, tech, minimalista, pulito (no look “stock photo”)
- Elemento brand: pattern geometrico/hexagon sottile

## HERO (1200×630)

Create a professional, modern tech illustration for an article about **[TOPIC]**.
Style: minimalist futurism, tech-forward, vibrant teals and blues, clean composition.
Subject: **[SPECIFIC_VISUAL_FROM_ARTICLE]**.
Include subtle geometric pattern and optional minimal typography overlay with **[KEYWORD_PRIMARY]** (readable).

## MID (800×500) — diagram/infographic

Create a clear technical infographic explaining **[KEY_CONCEPT]**.
Use labeled components, strong hierarchy, clean spacing, teal/white/dark accents.
Educational, single-visual explanation.

## SOCIAL TILE (1080×1080) — opzionale

Create an eye-catching social square about **[TOPIC]**.
High contrast, mobile-readable headline, reserved space for IlVantaggioAI.it logo.
