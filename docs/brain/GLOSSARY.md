# Glossario

- **Candidate**: una news candidata (`NewsCandidate`) prima della selezione.
- **Queue item**: elemento selezionato per diventare articolo.
- **Draft**: bozza lunga (markdown) generata da Gemini.
- **Ready**: articolo pronto per pubblicazione (HTML + meta + schema).
- **Idempotenza**: rilanciare uno step senza creare duplicati.
- **Freshness window**: finestra temporale massima delle fonti (target 7 giorni).
